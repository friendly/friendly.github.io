# Icons & Emojis Reference

Bootstrap Icons are used inline as `<i class="bi bi-NAME"></i>`.
They are available in all pages since the site uses Bootstrap themes (cosmo/darkly).

Emojis work anywhere in markdown or HTML — no class needed, just paste the character.

---
